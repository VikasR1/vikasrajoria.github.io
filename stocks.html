Stock Market Analysis

About Project

This project meant for analyzing the pattern and trends of shares based on their past performance. It uses python language and it's associated libraries like pandas, numpy , matplotlib, pandas-datareader and other helpful libraries and modules. Python allows to automate the day to day tasks which save time and reduce human error.

We have also used api key to fetch live data, which helps to get the record of a particular stock in a chosen time frame. Findings of this project describe the relationship between stock price and volume traded, after analyzing the data and chart we can see that price is inversely proportional to volume. Moreover, we can also observe the positive relationship between different stocks with the help of scatter-plot.
